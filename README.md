# 📸 Snapgram

Snapgram is a simplified Instagram-like web application that allows users to share photos, view other users' posts, and interact in a minimalist social media environment. This project was built as part of an academic submission using modern full-stack web technologies.

## 🚀 Features

- User Authentication (Sign up / Login)
- Upload and share images
- View all uploaded posts
- Responsive UI for desktop and mobile devices
- User-friendly photo feed layout

## 🛠️ Tech Stack

**Frontend:**
- HTML
- CSS
- JavaScript
- EJS (Embedded JavaScript Templates)

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB

**Other Tools:**
- Multer (for handling file uploads)
- Mongoose (MongoDB ODM)
